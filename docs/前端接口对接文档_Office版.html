前端接口对接文档（Office版）

更新时间：2026-04-07

接口前缀说明：当前服务按模块分组，实际路径为 /shaft/*。

1. 统一返回结构

1.1 普通返回

{
  "code": 200,
  "message": "success",
  "success": true,
  "data": {}
}

1.2 分页返回（data 为分页对象）

{
  "code": 200,
  "message": "success",
  "success": true,
  "data": {
    "total": 100,
    "list": [],
    "pageNo": 1,
    "totalSize": 100,
    "dataList": []
  }
}

1.3 告警处理 action 取值

confirm / observe / false_positive / close

1.4 告警状态 statusCode 取值

0 待确认，1 已确认，2 持续观察，3 误报，4 自动恢复

2. 告警模块（/shaft/alarm）

方法路径说明
GET/shaft/alarm/statistics告警统计
GET/shaft/alarm/list告警分页列表
GET/shaft/alarm/merge/list告警合并事件分页
GET/shaft/alarm/detail告警详情（query）
POST/shaft/alarm/handle单条处理（监测对象）
POST/shaft/alarm/batchHandle批量处理（监测对象）
GET/shaft/alarm/stat终端告警统计
GET/shaft/alarm/device/alarms终端告警分页
PUT/shaft/alarm/handle单条处理（终端）
PUT/shaft/alarm/batchHandle批量处理（终端）
GET/shaft/alarm/detail/{id}终端告警详情
GET/shaft/alarm/events事件列表
GET/shaft/alarm/alarm-rules规则列表
GET/shaft/alarm/alarm-rules/{id}规则详情
POST/shaft/alarm/alarm-rules新增规则
PUT/shaft/alarm/alarm-rules/{id}修改规则
POST/shaft/alarm/alarm-rules/{id}/enable启用规则
POST/shaft/alarm/alarm-rules/{id}/disable停用规则
DELETE/shaft/alarm/alarm-rules/{id}删除规则（逻辑删）

2.1 处理告警请求体示例

{
  "id": 1770000000000001,
  "action": "confirm",
  "handler": 10001,
  "handleRemark": "已确认"
}

2.2 批量处理请求体示例

{
  "alarmIds": [1770000000000001, 1770000000000002],
  "action": "false_positive",
  "handler": 10001,
  "handleRemark": "批量处理"
}

2.3 规则维护请求体示例

{
  "ruleName": "温度阈值规则",
  "bizType": "MONITOR",
  "alarmType": "TEMP_THRESHOLD",
  "scopeType": "GLOBAL",
  "scopeId": null,
  "level": 2,
  "thresholdValue": 70.0,
  "thresholdValue2": null,
  "durationSeconds": null,
  "enabled": 1,
  "remark": "默认规则"
}

3. 监测对象模块（/shaft/monitor）

方法路径说明
GET/shaft/monitor/detail/{id}监测对象详情
GET/shaft/monitor/statistics按 shaftType 统计告警数量
GET/shaft/monitor/raw-data原始测温数据查询

raw-data 时间参数 from/to 使用 ISO 时间格式，例如：2026-04-07T10:00:00

4. 设备模块（/shaft/device）

方法路径说明
GET/shaft/device/statistics设备在线统计
GET/shaft/device/detail/{id}设备详情
GET/shaft/device/ledger/list设备台账分页
GET/shaft/device/stat台账统计
GET/shaft/device/access/list接入清单分页
POST/shaft/device/access/confirm接入确认
GET/shaft/device/realtime/{deviceId}设备实时状态

4.1 接入确认请求体示例

{
  "deviceIds": [3001, 3002],
  "monitorId": 4001,
  "areaId": 5001,
  "orgId": 6001
}

5. 区域机构模块（/shaft/areaOrg）

方法路径说明
GET/shaft/areaOrg/tree机构/区域树
GET/shaft/areaOrg/template?type=org|area下载 CSV 模板
POST/shaft/areaOrg/import导入 CSV（multipart）

6. IoT 接入模块（/shaft/iot）

方法路径说明
POST/shaft/iot/reports/measure单条测温上报（全链路）
POST/shaft/iot/reports/alarm单条状态上报
POST/shaft/iot/data/batch批量测温
POST/shaft/iot/alarm/batch批量告警状态（当前返回 ack）

  "topic": "/TMP/shaft-dev-01_TMP_th01/Measure",
