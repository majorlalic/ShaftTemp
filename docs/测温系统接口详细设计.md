## 监测管理模块 Controller

### AlarmController - 告警相关

#### 1. 获取告警状态统计

**URL**: `GET /alarm/statistics`

**入参**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| monitorId | Long | 否 | 监测对象ID，不传则查询所有 |
| deviceId | Long | 否 | 终端ID，不传则查询所有 |
| startTime | DateTime | 否 | 开始时间 |
| endTime | DateTime | 否 | 结束时间 |

**出参**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "pendingConfirm": 25,
    "confirmed": 128,
    "observing": 12,
    "falseAlarm": 8,
    "total": 173
  }
}
```

#### 2. 获取告警列表

**URL**: `GET /alarm/list`

**入参**:
| 参数名 | 类型 | 必填 | 描述 | 示例 |
|--------|------|------|------|------|
| page | integer | 否 | 当前页码，默认1 | 1 |
| pageSize | integer | 否 | 每页条数，默认10 | 10 |
| deviceType | string | 否 | 设备类型（全部/配电柜局放/竖井测温/行波定位） | 全部 |
| substation | string | 否 | 变电站名称（输入框模糊匹配） | 奋进站 |
| terminalType | string | 否 | 终端类型（如：油色谱、GIS局放、开关柜测温等） | 油色谱 |
| alarmStartTime | string | 否 | 告警开始时间，格式：yyyy-MM-dd HH:mm:ss | 2023-10-11 00:00:00 |
| alarmEndTime | string | 否 | 告警结束时间，格式：yyyy-MM-dd HH:mm:ss | 2023-10-11 23:59:59 |
| status | string | 否 | 告警状态（待确认告警/持续观察告警） | 待确认告警 |
| alarmType | string | 否 | 告警类型（模糊匹配） | 内部空穴 |

**出参**:
```json

  "message": "success",
```

#### 5. 批量确认告警

**URL**: `PUT /alarm/batch/confirm`

**入参**:
```json
{
  "alarmIds": [1, 2, 3, 5],
  "action": "confirm",
  "handleRemark": "经现场确认，温度正常"
}
```

**出参**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "successCount": 4,
    "failCount": 0,
    "failIds": []
  }
}
```

**URL**: `PUT /alarm/{id}/confirm`

**入参**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| id | Long | 是 | 告警ID(path参数) |
| action | String | 是 | confirm/observe/false |
| handleRemark | String | 否 | 处理备注 |

**出参**:
```json

  "message": "success",
  "data": {
    "id": 1,
    "status": "confirmed",
    "confirmTime": "2024-03-18 14:30:25"
  }
}
```

| 值 | 说明 |
| :--- | :--- |
| 0 | 离线 |
| 1 | 在线 |
