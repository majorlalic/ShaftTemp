## 监测管理模块 Controller

### AlarmController - 告警相关

#### 1. 获取告警状态统计

**URL**: `GET /alarm/statistics`

**入参**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| areaId | Long | 否 | 区域ID,不传则查询所有, 选择的区域下所有的监控对象 |
| deviceId | Long | 否 | 终端ID，不传则查询所有 |
| startTime | DateTime | 否 | 开始时间 |
| endTime | DateTime | 否 | 结束时间 |

**出参**:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "pendingConfirm": 25, // 待确定
    "confirmed": 128, // 已确认
    "observing": 12, // 持续观察
    "falseAlarm": 8, // 误报
    "total": 173 // 合计
  }
}
```

#### 2. 获取告警列表

**URL**: `GET /alarm/list`

**入参**:

| 参数名 | 类型 | 必填 | 描述 | 示例 |
|--------|------|------|------|------|
| pageNum | integer | 否 | 当前页码，默认1 | 1 |
| pageSize | integer | 否 | 每页条数，默认10 | 10 |
| areaId    | Long    | 否   | 区域ID,不传则查询所有, 选择的区域下所有的监控对象 |                     |
| deviceId  | Long    | 否   | 终端ID，不传则查询所有                            |                     |
| startTime | string | 否 | 告警开始时间，格式：yyyy-MM-dd HH:mm:ss | 2023-10-11 00:00:00 |
| endTime | string | 否 | 告警结束时间，格式：yyyy-MM-dd HH:mm:ss | 2023-10-11 23:59:59 |
| status | string | 否 | 告警状态 |  |

**出参**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 12,
    "list": [
      {
        "id": 1, // 主键
        "alarmTime": "2023-10-11 13:22:23", // 告警时间
        "alarmCount": 7, // 告警总数
        "areaId": "", // 所属区域id
        "areaName": "", // 所属区域名称, 最好是全称
        "deviceId": "", // 设备id
        "deviceName": "", // 设备名称
        "deviceType": "", // 设备类型
        "alarmType": "", // 告警类型
        "alarmContent": "总烃产气速率过高，速率为13mL/d，告警阈值为12mL/d", // 告警内容
        "status": "", // 告警状态
      }
    ]
  }
}
```

#### 3. 获取告警合并事件列表

**URL**: `GET /alarm/merge/list`

**入参**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| alarmId | Long | 是 | 主告警ID |
| pageNum | Integer | 是 | 页码 |
| pageSize | Integer | 是 | 每页条数 |

**出参**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 5,
    "list": [
      {
        "id": 1001, // eventId
        "eventTime": "2024-03-18 10:23:15", // 
        "eventContent": "测试温度过高，温度为89，其阀值为80"
      }
    ]
  }
}
```

**入参**:

**出参**:

  "code": 200,
  "message": "success",
  "data": {
      "id": 1,
      "alarmCode": "AL202403180001",

      "status": "pending_confirm",
      "firstAlarmTime": "2024-03-18 10:23:15",

      "mergeCount": 3,

#### 5. 批量处理告警

**URL**: `Post /alarm/batchHandle`

**入参**:
```json
{
  "alarmIds": [1, 2, 3, 5],
  "action": "confirm",
  "handleRemark": "经现场确认，温度正常"
}
```

**出参**:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "successCount": 4,
    "failCount": 0,
    "failIds": []
  }
}
```

#### 6. 处理单条告警

**URL**: `Post /alarm/handle`

**入参**:

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| id | Long | 是 | 告警ID(path参数) |
| action | String | 是 | confirm/observe/false |
| handleRemark | String | 否 | 处理备注 |

**出参**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "status": "confirmed",
    "handleTime": "2024-03-18 14:30:25"
  }
}
```

## 数据字典补充说明

### 告警状态(status)
| 值 | 说明 |
| :--- | :--- |
| pending_confirm | 待确认 |
| confirmed | 已确认 |
| observing | 持续观察 |
| false_alarm | 误报 |

### 告警类型(alarm_type)
| 值 | 说明 | 所属模块 |
| :--- | :--- | :--- |
| over_temp | 超温告警 | 监测管理 |
| diff_temp | 差温告警 | 监测管理 |
| rate_temp | 升温速率告警 | 监测管理 |
| offline | 离线告警 | 终端管理 |
| broken | 断纤告警 | 终端管理 |

### 在线状态(online_status)
| 值 | 说明 |
| :--- | :--- |
| 0 | 离线 |
| 1 | 在线 |
| 2 | 故障 |
